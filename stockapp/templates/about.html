About

Welcome to Stock Price Prediction App! This is an application built using machine learning algorithms that predicts stock prices based on historical data.

The Stock Price Prediction App was created by Uttam Patel as a personal project to learn and practice machine learning skills. It is open source and available on GitHub at https://github.com/Uttampatel1.

Technologies Used

  • Python 3.9
  • Flask 2.1.0
  • Pandas 1.3.3
  • Scikit-learn 1.0
  • HTML 5
  • CSS 3
  • Bootstrap 5.1.0

Features

  • User authentication
  • Predict stock prices based on historical data
  • Interactive data visualization using Plotly.js
  • Download predicted stock prices as a CSV file
  • Responsive design

Getting Started

To use the Stock Price Prediction App, follow these steps:

  1. Clone the repository: git clone -------
  2. Navigate to the project directory: cd stock-price-prediction-app
  3. Create a virtual environment: python3 -m venv venv
  4. Activate the virtual environment: source venv/bin/activate
  5. Install the required packages: pip install -r requirements.txt
  6. Run the app: flask run
  7. Open your web browser and go to http://localhost:5000
